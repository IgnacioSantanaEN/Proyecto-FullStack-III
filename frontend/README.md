# FrontEndExamen

Proyecto React + Vite para el panel administrativo de SmartLogix.

## Estructura

- `src/App.jsx` — raíz de la aplicación.
- `src/main.jsx` — punto de entrada de Vite.
- `src/pages/` — vistas de la aplicación.
- `src/components/` — componentes reutilizables y layout.
- `src/constants/` — datos mock.
- `src/styles/` — tokens de diseño y theme.
- `src/utils/` — utilidades puras.

## Ejecutar

1. `npm install`
2. `npm run dev`

## Build

- `npm run build`
- `npm run preview`
